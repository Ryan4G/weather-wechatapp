# 天气预报小程序
一个实现基本天气情况查询的微信小程序。目前能够实现以下功能：
1. 查看当前地理位置的实时天气预报及未来三天天气情况；
2. 查看管理多个城市的天气情况；
3. 添加符合输入关键字的国内城市，调整已添加城市的显示顺序及移除城市；
4. 清除当前小程序内的缓存数据。

## 界面布局
1. 主页显示天气情况，主要使用了swiper进行多城市天气查看切换；
2. 管理城市页面可跳转至“调整/移除顺序界面” 和 “查询添加城市界面”；
3. 缓存设置界面用于清除当前程序缓存数据。

## 数据来源
数据来源于：和风天气。
图片来源于：和风天气图标包及小米论坛。
